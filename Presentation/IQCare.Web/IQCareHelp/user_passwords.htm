User Passwords

  

For IQCare, there are no special password requirements or restrictions. Special characters can be used within the password. You should have your own IQCare Username and password and should not share these access credentials with other users. Your IQCare Administrator will create a Username and Password for you. Once you have your Username and Password, you should change your password so that only you know it.
