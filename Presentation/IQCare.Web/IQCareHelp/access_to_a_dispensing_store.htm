Access to a Dispensing Store

  
Store type

A store must be designated as a dispensing store in order for you to dispense drugs from that store. If you attempt to access a store that is not set up as a dispensing store, the system will deny access. To set up a dispensing store, see Store Detail.

 

Access rights

There is access security set up for every store. When you initiate dispensing, IQCare Management asks you to select a store. If you have access to the store, you can enter the store for dispensing. If you do not have access, the system will display a message letting you know that you don't have access rights. To set the access rights to a store, see Store User Linking.

 

To access a dispensing store

1.Select PM/SCM / Dispense Drugs from the navigation bar.

 

2.Select a user store from the select list.

 

3.If you have access to the dispensing store you selected, the system will display the Dispensing window, defaulted to the Find Patient page. If you do not, the system will display an access error.
