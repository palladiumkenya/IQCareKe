Edit an Appointment Via the Patient Home Page

  

1.Navigate to the patient’s home page

2.Click on the Schedule Appointment link found in the Patient level navigation. The patient’s appointment history is displayed.

 

 

3.The existing appointment information is populated and is editable. Make changes to the date, purpose or provider

 

4.Click Update to save the changes

5.The patient’s schedule history is updated
